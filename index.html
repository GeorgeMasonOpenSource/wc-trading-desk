What changed (vs prior version): Team-level goals model is now bookmaker-direct for the entire group stage — CS prob is read from the DraftKings Team Clean Sheet market (130/144 team-MDs), team xG is solved from the Team Total Goals (over-1.5) market, and group advance is computed from devigged 3-way moneylines. The Poisson approximation we used before is bypassed where bookie odds exist.

What's still modeled (the honest gap): Player-level shares (gs, as, tk90, cc90, sot90, pen). The bookies do price these (anytime scorer at +200, anytime assist at +400, etc.) but ESPN's free feed gives only team-level props, not player-level. Per the no-paid-APIs constraint, those remain on price-tier/position heuristics. So when the app says "Yamal expected involvement", the team xG and CS context is bookie-direct, but his share of that team's goals is still proxy.

Source check: Open ESPN scoreboard for any WC fixture → "Match Predictor" tab. Our cs_prob and team xG should track DraftKings' published lines to within a few percent (devig drift only).

xPts formula 
xPts = appearance_pts                          // P(60+)×2 + P(brief sub)×1
     + goals_exp × goal_pts[pos]                // GK=9 DEF=7 MID=6 FWD=5
     + assists_exp × 3
     + adjMin × team_cs_prob × cs_pts[pos]      // DEF/GK=5, MID=1 (60+ min req)
     + tk90 / 3  × adjMin                       // MID only
     + cc90 / 2  × adjMin                       // MID chances created
     + sot90 / 2 × adjMin                       // FWD shots on target
     + pen90 × 2 × adjMin                       // P(winning a pen) × 2
     + 0.001 × adjMin × 9                       // GK goal (rare)
     + 0.10 × 0.22 × adjMin × 3                 // GK pen save EV
     + bonus_exp + fk_bonus_exp + scouting_bonus_ev
     - 0.18 × adjMin × card_modifier            // yellow EV
     - 0.018 × adjMin × 2                       // red EV
     - max(0, λ_play - (1 - e^-λ_play))         // -1 per goal above 1st (DEF/GK)

minutes_dist = [start_p × 60min_p × 90min_p, sub_p, bench_p, injury_p]
goals_exp     = team_xG × player_goal_share × adj_minutes
assists_exp   = team_xG × player_assist_share × adj_minutes
team_xG       = base_xG × (att / opp_def) × (1 + hot_mod) × (1 - heat_mod)
cs_prob       = exp(-team_xGA × adj_minutes / 90)

Design Principles

  • Never recommend action unless EV clears threshold.
  • Always compare against doing nothing.
  • Separate EV from risk, projection from confidence.
  • Surface stale assumptions; every line is inspectable.
  • Deterministic. No ML / LLM / black-box in decision path.
  • Tournament-aware: weights group stage less than knockouts in horizon > 3.
  • Reduce confidence for hot streaks; long-run priors dominate.
  • 3-per-country cap enforced everywhere (group stage).
